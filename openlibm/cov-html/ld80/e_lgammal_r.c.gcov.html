      32                 :             :  * Method:
      33                 :             :  *   1. Argument Reduction for 0 < x <= 8
      34                 :             :  *      Since gamma(1+s)=s*gamma(s), for x in [0,8], we may
      35                 :             :  *      reduce x to a number in [1.5,2.5] by

      38                 :             :  *              lgamma(7.3) = log(6.3) + lgamma(6.3)
